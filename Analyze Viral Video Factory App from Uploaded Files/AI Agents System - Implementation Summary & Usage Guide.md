## 2. How to Use the AI Agents Hub

The new **AI Agents Hub** is accessible from the sidebar under the "Creator Tools" section. Here’s how to use it:

1.  **Select an Agent**: Choose from the four available agents based on your needs.
2.  **Choose an Action**: Each agent has a set of specific actions you can perform.
3.  **Provide Input**: Enter your content, topic, or script for analysis.
4.  **Run the Agent**: Click the "Run AI Agent" button to process your request.
5.  **View Results**: The results will appear in the "Recent Results" section.

**Example Use Cases:**
- **Find viral trends**: Use the Content Strategy Agent to research trending topics in your niche.

## 3. N8N Integration for Automated Posting

The Brand Consistency Agent is integrated with N8N for automated multi-platform posting. Here’s how it works:

1.  **Configure N8N Webhook**: Add your N8N webhook URL to your brand kit settings.
2.  **Run Brand Analysis**: The Brand Consistency Agent will analyze your content for brand alignment.
3.  **Trigger N8N Workflow**: If the brand score is above the threshold (default: 80), the agent will trigger your N8N workflow with the optimized content and posting schedule.

This creates a seamless workflow from content creation to multi-platform distribution, ensuring perfect brand consistency across all channels.
